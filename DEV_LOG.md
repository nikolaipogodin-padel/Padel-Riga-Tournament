# Известные баги (BUG-BOARD)
- BUG-001: Связь в LIVE-матчах идет через String (имена), а не через User ID. При смене имени в профиле игрок «теряет» свои матчи.
- BUG-002: Статус FINISHED никогда не устанавливается (отсутствует логика проверки завершения всех матчей).
- BUG-003: В localStorage хранятся дублирующиеся данные демо-пользователей и реальных профилей.
